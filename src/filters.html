Filters

Filters in Eleventy

Filters in Eleventy are added to the .eleventy.js config file. They are JavaScript functions that accept content, modify it, then return it to be display in place of the original.

Eleventy does lack certain formatting features that cna be found in other templating platforms, date formatting for example. A JavaScript function can be added as a filter to provide that formatting.

Eleventy Config Object

To make Eleventy content available to the config file, the module.exports must be changed into a function. It can then accept an Eleventy config object, in this manner : 

module.exports = (function(eleventyConfig) {

}
